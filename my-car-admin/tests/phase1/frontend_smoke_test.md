# Phase 1 前端冒烟测试清单

## 测试目标

验证 Phase 1 前端基础功能可用：登录页、路由守卫、基础布局、Dashboard。

## 测试步骤

1. 启动前端服务：`npm run dev`
2. 浏览器访问 `http://127.0.0.1:5174`
3. 验证未登录自动跳转 `/login`
4. 输入管理员账号密码并登录
5. 验证进入主页并显示 `数据看板`
6. 点击右上角 `退出登录`
7. 验证返回登录页

## 预期结果

- 退出登录后 Token 被清理
