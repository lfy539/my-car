# Phase 2 UI 测试用例

## 用例 1：品牌管理

1. 进入 `品牌管理` 页面
2. 点击 `新增品牌` 并保存
3. 在列表执行 `编辑` 并保存
4. 在列表执行 `删除`
5. 预期：增删改查均成功，列表实时刷新

## 用例 2：轮播图管理

1. 进入 `轮播图管理` 页面
2. 新增一条轮播图记录
3. 编辑该记录的标题
4. 删除该记录
5. 预期：记录状态与内容更新正确

## 用例 3：壁纸管理

1. 进入 `壁纸管理` 页面
2. 新增壁纸（包含 `coverUrl`、`originUrl`、`brandId`）
3. 编辑壁纸标题与标签
4. 删除壁纸
5. 预期：列表渲染正确、操作成功提示正确

## 用例 4：音效管理

1. 进入 `音效管理` 页面
2. 新增音效（包含 `audioUrl`、`brandId`）
3. 编辑音效类型和时长
4. 删除音效
5. 预期：增删改查完成且页面无报错
